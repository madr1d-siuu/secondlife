## Функциональность

### Для всех пользователей
- Просмотр каталога объявлений с фильтрами по категориям и городам
- Поиск по названию и описанию товара
- Просмотр фото и видео (YouTube/Vimeo) в карточках товаров
- Страницы «О нас», «Поддержка» с FAQ

### Для зарегистрированных пользователей
- Регистрация и авторизация
- Создание, редактирование и удаление собственных объявлений
- Загрузка фотографий товаров
- Добавление ссылок на видео (YouTube, Vimeo, MP4)
- Избранное
- Чат с продавцом
- Бронирование товаров с уведомлением продавца
- Раздел «Мои заказы» с возможностью отмены брони
